# 数据类型

JavaScript 中有 8 种基本的数据类型。

## 动态类型

可以将任意类型的值存入变量，例如，一个变量可以在前一刻是字符串，下一刻存储一个数字：

```javascript
let message = "hello";
message = 123456;
```

允许这种操作的编程语言，被称作**动态语言**。意思是虽然编程语言中有不同的数据类型，但是你定义的变量并不会在定义后被限制为某一数据类型。

## Number 类型

```javascript
let n = 123;
n = 12.345;
```

**number 类型**代表整数和浮点数。

数字可以有加减乘除等操作。除了常规的数字，还包括所谓的"特殊数值"也属于这种类型：

| 特殊值 | 含义 |
|--------|------|
| `Infinity` | 数学概念中的无穷大，比任何数字都大的特殊值 |
| `-Infinity` | 负无穷大 |
| `NaN` | 计算错误，不正确的或未定义的数学操作所得到的结果 |

### Infinity

`Infinity` 代表数学概念中的无穷大。可以通过除以 0 来得到它：

```javascript
alert(1 / 0);
```

也可以在代码中直接使用它：

```javascript
alert(Infinity);
```

### NaN

`NaN` 代表一个计算错误，它是一个不正确的或者一个未定义的数学操作所得到的结果，比如：

```javascript
alert("not a number" / 2);
```

> [!warning] NaN 的粘性
> 任何对 `NaN` 的进一步数学运算都会返回 `NaN`：
>
> ```javascript
> alert(NaN + 1);                 // NaN
> alert(3 * NaN);                 // NaN
> alert("not a number" / 2 - 1);  // NaN
> ```
>
> 所以，如果在数学表达式中有一个 `NaN`，会传播到最终结果。
>
> 只有一个例外：`NaN ** 0` 结果为 `1`。

## BigInt 类型

在 JavaScript 中，**number 类型**无法安全地表示大于 `(2^53 - 1)`（即 `9007199254740991`）或小于 `-(2^53 - 1)` 的整数。
