# 工具

记录我会用到的系统工具及其介绍和下载地址。

## Ventoy

**Ventoy** 是一款开源的 USB 启动盘制作工具。与传统工具（如 Rufus）每次只能写入一个镜像不同，Ventoy 的特点是：只需安装一次到 U 盘，之后只需将 ==ISO/WIM/IMG/VHD(x)/EFI== 镜像文件直接**复制**到 U 盘即可启动，无需反复格式化。

> [!tip] 核心优势
> 一个 U 盘可以同时存放多个系统镜像（Windows、Linux、PE 等），启动时 Ventoy 会显示菜单让你选择启动哪个。

### 下载地址

- 官网：[https://www.ventoy.net](https://www.ventoy.net)
- GitHub：[https://github.com/ventoy/Ventoy](https://github.com/ventoy/Ventoy)

## Rufus

**Rufus** 是一款 Windows 平台上的 USB 启动盘制作工具，以其**速度快、体积小**著称。它可以将 ISO 镜像写入 U 盘，支持 ==BIOS== 和 ==UEFI== 两种启动模式，广泛用于安装 Windows 和 Linux 系统。

> [!notice]
> Rufus 仅支持 **Windows** 平台。Linux 用户可使用 Ventoy 或 `dd` 命令替代。

### 下载地址

- 官网：[https://rufus.ie](https://rufus.ie)
- GitHub：[https://github.com/pbatard/rufus](https://github.com/pbatard/rufus)

## GParted

**GParted**（GNOME Partition Editor）是一款图形化磁盘分区管理工具。它可以创建、删除、调整、移动、复制分区，支持 ==ext2/3/4==、==NTFS==、==FAT16/32==、==Btrfs==、==XFS== 等多种文件系统。

> [!warning]
> 分区操作有数据丢失风险，操作前务必**备份重要数据**。

GParted 提供两种使用方式：

1. **Live ISO** —— 制作成启动 U 盘，在操作系统之外进行分区操作（适合调整系统盘分区）
2. **Linux 包管理器安装** —— 在已有 Linux 系统中直接安装使用

```shell
sudo pacman -S gparted
```

### 下载地址

- 官网：[https://gparted.org](https://gparted.org)
- Live ISO：[https://gparted.org/download.php](https://gparted.org/download.php)
